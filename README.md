# Simple Snake Game

파이썬과 Pygame으로 구현한 뱀 게임입니다.

## 기능

- **두 가지 게임 모드**
  - Classic 모드: 뱀이 자동으로 이동, 방향키로 방향 전환
  - Relaxed 모드: 키 입력시에만 이동, 잘못된 이동은 무시
- **보드 크기 설정**: 6x6 ~ 32x32 셀 사이에서 자유롭게 조정
  - 일반 조정: 1칸씩 증감
  - 빠른 조정: Shift + 방향키로 10칸씩 증감
- **고정 윈도우**: 800x800 픽셀 윈도우에 게임판이 중앙 정렬
- **점수 시스템**: 시작 1점, 먹이 하나당 +1점
- **충돌 감지 및 게임 종료**
  - Classic 모드: 벽이나 자기 몸에 부딪히면 게임 오버
  - Relaxed 모드: 잘못된 입력 무시, Q 키로 직접 종료 가능
- **승리 조건**: 보드를 완전히 채우면 게임 클리어
- **게임 종료 화면**: 재시작 또는 메인 메뉴 선택 가능
- **입력 처리 개선**: 빠른 연속 입력 시에도 안전한 방향 전환

## 설치 및 실행

### 필요 사항
- Python 3.10 이상
- uv 패키지 매니저

### 실행 방법

```bash
# 최초 1회 의존성 설치
uv sync

# 게임 실행
uv run python snake_game.py
```

## 게임 조작법

### 시작 화면
- **방향키 (↑ ↓)**: 커서 이동
- **방향키 (← →)**: 보드 크기 조정
  - 일반: 1칸씩 증감
  - Shift + 좌우: 10칸씩 증감
- **ENTER**: 게임 시작

### 게임 중
- **방향키 (↑ ↓ ← →)**: 뱀 이동 또는 방향 전환
- **Q**: 게임 오버

### 게임 오버 / 클리어 화면
- **방향키 (↑ ↓)**: 메뉴 선택
- **ENTER**: 선택 확인
- **R**: 즉시 재시작
- **창 닫기**: 게임 종료

## 게임 규칙

### Classic 모드
1. 뱀이 자동으로 이동, 방향키로 방향만 변경
2. 빨간색 먹이를 먹으면 뱀이 즉시 성장하고 점수 +1
3. 벽이나 자기 몸에 부딪히면 게임 오버
4. 뱀은 마지막 실제 이동 방향의 반대로 회전할 수 없음
5. 보드를 완전히 채우면 게임 클리어

### Relaxed 모드
1. 키를 누를 때만 한 칸씩 이동
2. 빨간색 먹이를 먹으면 뱀이 즉시 성장하고 점수 +1
3. 벽이나 자기 몸으로 가는 입력은 무시됨
4. 보드를 완전히 채우면 게임 클리어

### 공통 규칙
- 게임 시작 시 뱀은 항상 보드 중앙에 생성
- 먹이는 뱀과 겹치지 않는 위치에 생성
- 먹이를 먹는 순간 뱀이 즉시 늘어남
- Q 키로 직접 게임 오버 가능

## 코드 구조
