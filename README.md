# Simple Snake Game

파이썬과 Pygame으로 구현한 클래식 뱀 게임입니다.

## 기능

- 그리드 기반 뱀 이동
- 먹이 섭취 시 뱀 성장
- 벽 및 자기 몸과의 충돌 감지
- 점수 시스템
- 게임 오버 및 재시작

## 설치 및 실행

### 필요 사항
- Python 3.10 이상
- uv 패키지 매니저

### 실행 방법

```bash
# 의존성 설치 (최초 1회)
uv sync

# 게임 실행
uv run python snake_game.py
```

## 게임 조작법

- **방향키 (↑ ↓ ← →)**: 뱀 이동
- **SPACE**: 게임 오버 후 재시작
- **ESC / 창 닫기**: 게임 종료

## 게임 규칙

1. 빨간색 먹이를 먹으면 뱀이 성장하고 점수가 10점 증가합니다
2. 벽이나 자기 몸에 부딪히면 게임 오버
3. 뱀은 반대 방향으로 즉시 회전할 수 없습니다

## 기술 스택

- **언어**: Python 3.10
- **라이브러리**: Pygame 2.6.1
